# 상속

상속(Inheritance)은 기존 클래스의 속성과 메서드를 물려받아 새로운 클래스를 만드는 기능이다.

## 사용 이유

상속을 사용하면 이미 작성한 코드를 재사용할 수 있으며, 공통 기능을 한 곳에서 관리할 수 있다.

예를 들어 사람(Person) 클래스가 있다고 하자.

```python
class Person:
    def __init__(self, name):
        self.name = name

    def introduce(self):
        print(f"안녕하세요. 저는 {self.name}입니다.")
```

학생(Student)도 이름을 가지고 자기소개를 할 수 있다. 하지만 학생만의 기능도 필요할 수 있다.

이때 상속을 사용하면 기존 Person 클래스를 재사용하면서 Student 클래스를 만들 수 있다.

## 상속 문법

상속은 클래스 이름 뒤의 괄호에 부모 클래스를 작성하여 사용한다.

```python
class 자식클래스(부모클래스):
    ...
```

예를 들어 Student가 Person을 상속받는다면 다음과 같이 작성한다.

```python
class Person:
    def introduce(self):
        print("안녕하세요.")

class Student(Person):
    ...
```

이제 Student는 introduce 메서드를 사용할 수 있다.

```python
s = Student()
s.introduce()
```

실행 결과

```text
안녕하세요.
```

## 생성자 상속

메서드뿐만 아니라 생성자도 상속된다.

```python
class Person:
    def __init__(self, name):
        self.name = name

class Student(Person):
    pass

s = Student("Kim")
print(s.name)
```

실행 결과

```text
Kim
```

Student에 생성자가 없기 때문에 Person의 생성자가 사용된다.

## super()

자식 클래스에서 생성자를 직접 만들 경우 부모 생성자는 자동으로 호출되지 않는다.

```python
class Student(Person):
    def __init__(self, name, grade):
        self.name = name
        self.grade = grade
```

이 경우 부모 생성자의 기능을 직접 호출해야 한다.

이를 위해 super()를 사용한다.

```python
class Student(Person):
    def __init__(self, name, grade):
        super().__init__(name)
        self.grade = grade
```

super()는 부모 클래스의 메서드에 접근할 때 사용한다.

## 메서드 오버라이딩

자식 클래스는 부모의 메서드를 재정의할 수 있다.

이를 오버라이딩(Overriding)이라고 한다.

```python
class Person:
    def introduce(self):
        print("사람입니다.")

class Student(Person):
    def introduce(self):
        print("학생입니다.")
```

```python
s = Student()
s.introduce()
```

실행 결과

```text
학생입니다.
```

같은 메서드를 호출했지만 객체의 종류에 따라 다른 동작이 수행된다.

## isinstance

상속 관계에서는 isinstance를 사용하여 객체의 타입을 확인할 수 있다.

```python
class Person:
    pass

class Student(Person):
    pass

s = Student()

print(isinstance(s, Student))
print(isinstance(s, Person))
```

실행 결과

```text
True
True
```

Student 객체는 Student이면서 동시에 Person이기 때문이다.

## 기본 자료형 상속받아 쓰기

리스트를 상속받아 0을 포함한 2의 배수인 인덱스만 가저오는 클래스를 만들 수 있다.

```python
class EvenList(list):
    def __getitem__(self, index):
        return super().__getitem__(index * 2)

nums = EvenList(x for x in range(7))

print(nums[0])  # 0
print(nums[1])  # 2
print(nums[2])  # 4
```

이처럼 메직메서드를 오버라이딩하여 기본 기능을 ~~재밌게~~ 바꿀 수 있다.

그런데 슬라이싱은 불가능하다. 그 이유는 `__getitem__` 이 인덱싱만 하는것이 아닌 슬라이싱도 하기 때문이다.

```python
class Test:
    def __getitem__(self, index):
        print(type(index))

t = Test()

t[1]
t[1:5]
```

출력

```text
<class 'int'>
<class 'slice'>
```

그렇기에 슬라이싱을 하고싶다면 isinstance 등을 이용하여 분기처리를 해주어 코드를 작성하면 된다.
