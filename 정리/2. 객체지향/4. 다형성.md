# 다형성

다형성(Polymorphism)은 같은 메서드 호출이라도 객체의 종류에 따라 다른 동작을 수행하는 것이다.

```python
class God:
    def speak(self):
        print("나는 신이다")

class Cat:
    def speak(self):
        print("야옹")

class Robot:
    def speak(self):
        print("로봇임")


li = [God(), Cat(), Robot()]

for o in li:
    o.speak()
```

실행 결과

```text
나는 신이다
야옹
로봇임
```

위 코드에서 speak()라는 같은 메서드를 호출했지만 객체의 종류에 따라 다른 결과가 출력된다. 이것이 다형성이다.

파이썬에서는 객체의 실제 타입보다 필요한 메서드가 존재하는지를 더 중요하게 생각한다. 이를 덕 타이핑(Duck Typing)이라고 한다.

"오리처럼 걷고 오리처럼 소리 내면 오리로 취급한다."

즉, 객체가 어떤 클래스인지보다 필요한 기능을 제공하는지가 중요하다.

## 마지막
